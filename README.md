# Nix Flakes


## Usage for [Devbox](https://www.jetify.com/devbox)

```bash
devbox add github:cytopia/nix-flakes#terraform-1-1-0

devbox add github:cytopia/nix-flakes#terragrunt-0-98-0

devbox add github:cytopia/nix-flakes#terragrunt-atlantis-config-1-21-1
```

## Update this repo
```
rm flake.lock && nix flake lock
```
